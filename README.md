# Unity-Mobile-BallDropPuzzle

Gameplay Video Link : https://www.youtube.com/watch?v=jQkyvtUIStY&list=PLeUQ9Opn9i7PpZXLhEPG_rDb0LjjYapMA&index=13

- Physics based puzzle gameplay
- Interactive environmental elements (bombs, moving platforms)
- Rope cutting mechanic featuring dynamically connected hinge joints linked to the ball
- Collectible star system
- 8 handcrafted playable levels
- Animated UI transitions using DOTween
- Designed with mobile gameplay in mind
